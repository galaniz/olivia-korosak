# Duration

## getDuration  

**<code>getDuration(seconds?: number, words?: boolean): string</code>**  

Convert seconds to text (time or words).

### Parameters  
- **`seconds`** <code>number</code> optional  
Default: `0`  
- **`words`** <code>boolean</code> optional  
Default: `false`

### Returns  

<code>string</code>

## getDurationSeconds  

**<code>getDurationSeconds(time: string): number</code>**  

Number of seconds from time (hh:mm:ss).

### Parameters  
- **`time`** <code>string</code> required  
Default: `00:00`

### Returns  

<code>number</code>
