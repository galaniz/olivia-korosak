# Olivia Korosak  

Static site for Olivia Korosak using [Formation](https://github.com/galaniz/formation) and [Formation Static](https://github.com/galaniz/formation-static).

## Installation

```shell
pnpm install --optional
```

## Local Development

To launch the local server:

```shell
pnpm clean
pnpm build
pnpm start
```
